# Distribuidora-Pitangueiras
O código se trata de uma landing page responsiva para a Distribuidora Pitangueiras, focada em polpas e frutas 100% orgânicas vindas da agricultura familiar. Feita com HTML e W3.CSS, possui um menu lateral e seções bem estruturadas sobre o compromisso da marca, rede de parceiros locais, fotos de comércios e rodapé com dados comerciais de contato.
